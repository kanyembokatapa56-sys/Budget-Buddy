# Budget Buddy Android Application

## Personal Finance Tracking App

Budget Buddy is a Kotlin-based Android budgeting application designed to help users record expenses, manage spending categories, monitor budgets and track financial goals from one mobile app.

The project was built to solve a simple problem: many people struggle to keep track of where their money goes when expenses, categories and goals are not organised in one place. Budget Buddy brings these functions together in a structured mobile experience.

## Project Overview

Budget Buddy allows users to:

* Create and access a personal account
* Record expenses and transactions
* Organise spending by category
* View budget-related information
* Monitor financial goals
* Navigate between multiple app screens
* Store user and budget data using Firebase

The goal was to create a practical Android application that supports basic personal finance management while demonstrating mobile development, data storage, authentication and user-focused design.

---

## Problem the Project Solves

Tracking expenses manually can become confusing when users do not have one place to manage their budget, categories and financial goals.

Budget Buddy addresses this by giving users a mobile platform where they can record transactions, manage categories and view their financial information in a more organised way.
